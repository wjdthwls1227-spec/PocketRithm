# Google OAuth 로그인 화면 브랜딩 변경 가이드

## 🎨 "kxoydaogtvkmjpoztnid.supabase.co 서비스로 로그인" → "포켓리즘으로 로그인" 변경

Google 로그인 화면에 표시되는 서비스 이름을 변경하려면 Google Cloud Console의 **OAuth 동의 화면** 설정을 수정해야 합니다.

## 📝 단계별 가이드

### 1단계: Google Cloud Console 접속

1. **Google Cloud Console 접속**
   - https://console.cloud.google.com
   - 프로젝트 선택 (PocketRithm)

2. **API 및 서비스 → OAuth 동의 화면**
   - 왼쪽 사이드바에서 **"API 및 서비스"** 클릭
   - **"OAuth 동의 화면"** 메뉴 클릭
   - 또는 직접 링크: https://console.cloud.google.com/apis/credentials/consent

### 2단계: 앱 정보 수정

#### 기본 정보 설정

1. **사용자 유형 선택**
   - **"외부"** 선택 (일반 사용자에게 공개)
   - 또는 **"내부"** (Google Workspace 조직 내부용)
   - **"만들기"** 클릭

2. **앱 정보 입력**
   - **앱 이름**: `포켓리즘` 또는 `PocketRithm`
     - 이 이름이 Google 로그인 화면에 표시됩니다
   - **사용자 지원 이메일**: `wjdthwls12@naver.com` (또는 본인 이메일)
   - **앱 로고** (선택사항): 
     - 120x120px 이상의 정사각형 이미지
     - PNG 또는 JPG 형식
     - 로고를 추가하면 더 전문적으로 보입니다

3. **앱 도메인** (선택사항)
   - **앱 홈페이지**: `https://pocket-rithm.vercel.app` (또는 실제 도메인)
   - **개인정보처리방침 링크**: `https://pocket-rithm.vercel.app/privacy`
   - **서비스 약관 링크**: `https://pocket-rithm.vercel.app/terms`

4. **승인된 도메인** (자동 추가됨)
   - OAuth 클라이언트 ID에 등록한 도메인은 자동으로 추가됩니다
   - `pocket-rithm.vercel.app`
   - `kxoydaogtvkmjpoztnid.supabase.co`

### 3단계: 범위(Scopes) 설정

1. **범위 추가 또는 확인**
   - **"+ 범위 추가 또는 삭제"** 클릭
   - 다음 범위가 선택되어 있는지 확인:
     - `email` - 사용자 이메일 주소
     - `profile` - 기본 프로필 정보 (이름, 프로필 사진)
   - **"업데이트"** 클릭

### 4단계: 테스트 사용자 추가 (외부 앱인 경우)

앱이 **"외부"**로 설정되어 있고 아직 검토를 받지 않은 경우:

1. **테스트 사용자** 섹션
2. **"+ 사용자 추가"** 클릭
3. 본인 이메일 주소 추가: `wjdthwls1227@gmail.com`
4. **"저장"** 클릭

### 5단계: 저장 및 확인

1. **"저장 후 계속"** 또는 **"저장"** 클릭
2. 변경사항이 적용되는 데 **5분에서 몇 시간** 걸릴 수 있습니다

- 특수 문자 사용 가능
- 한글 사용 가능
